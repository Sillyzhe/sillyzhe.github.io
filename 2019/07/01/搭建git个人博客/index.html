git部署个人博客

从事前端工具已经快一年了，这期间踩了不少坑，自以为记忆力不错可以记住之前犯的错。结果现实狠狠的扇了自己一巴掌，遇到过的坑已经忘得一干二净了。为了能提高自己，也为了以后可以有地方查阅到自己解决过的问题，所以打算开一个技术博客。
但是目前还没有精力自己买域名部署服务器（还是自己太垃圾了），所以利用git搭建一个个人博客就足够了。
我采用的方式是git page + hexo

所需工具

  • 创建git账号
  • 安装git bash
  • 安装node.js

账号创建及软件安装请自行百度或谷歌（很简单，小白用户也可尝试）

1、创建git仓库

注册完github后点击右上角的加好创建new repository

Respository name中的username.github.io 的username 一定与前面的Owner 一致，记住username后面会用到。
因为我之前创建过了，所以这里提示我当前账号已经存在该项目

2、安装hexo

1 执行以下命令安装hexo

$ sudo npm install hexo-cli -g

2 初始化博客

安装完成后我们就可以初始化博客了，如果想要把文件放到某个目录下先cd到指定目录下在执行下方命令，如果没有指定目录默认是的你的个人主目录下。（建议将初始化博客新建一个仓库，便于后期多端维护）

初始化好进入项目目录文件，打开git bash npm install
这时可以先本地跑一下这个项目，看看发布后的网站长什么样子

看到这个页面就是初始化成功了 3 部署

打开项目文件里的_config.yml文件

拉到最下面找到deploy的选项
